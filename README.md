# Audio editor
A simple audio editor in HTML/CSS/JS without any third-party libraries. Works in the browser. Suitable for cases when you need to cut a track, change its speed, pitch, or volume.

**Launch Audio editor** ➡️ https://limafresh.github.io/AudioEditor/
